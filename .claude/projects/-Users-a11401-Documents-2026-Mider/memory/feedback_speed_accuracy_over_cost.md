비용보다 속도와 정확도가 더 중요하다.

**Why:** LLM API 호출 비용 절감을 위해 2-Pass로 나누는 전략이 오히려 응답 시간을 160초까지 늘리는 병목이 됨. 사용자는 빠른 결과와 정확한 분석을 원함.

**How to apply:** LLM 호출 최적화 시 호출 횟수 감소(속도)와 attention 분산 방지(정확도)를 비용 절감보다 우선. mini 모델 선별 단계를 생략할 수 있으면 생략.
