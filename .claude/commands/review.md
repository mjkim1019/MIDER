코드 리뷰 전문가 에이전트를 호출하여 현재 변경사항을 리뷰합니다.

## Step 1: 변경사항 수집
1. `git diff main...HEAD`로 전체 변경 내역을 확인한다
2. 변경된 파일 목록과 diff 내용을 정리한다

## Step 2: 리뷰 에이전트 호출
Task tool (subagent_type: general-purpose)로 리뷰 에이전트를 호출한다.

리뷰 에이전트 프롬프트:
"""

- Pydantic v2 모델, JSON Mode 출력
- print() 금지 (logging/rich 사용)
- Agent는 코드 수정 불가 (제안만)

[리뷰 관점]

{diff 내용}

## Step 3: 결과 보고
